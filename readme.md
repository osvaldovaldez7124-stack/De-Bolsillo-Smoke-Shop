## 📌 Descripción

**De Bolsillo Smoke Shop** es una tienda en línea simulada para productos recreativos sostenibles. El proyecto fue desarrollado como trabajo final para la materia de **Desarrollo Web** con las siguientes funcionalidades:

- 🔐 Sistema de autenticación (registro e inicio de sesión)
- 🛒 Carrito de compras interactivo
- 💳 Vinculación de tarjetas (simulada)
- 📦 Catálogo de productos (Grinders, Bongs, Pipas)
- 👤 Perfil de usuario con historial de compras
- 🌱 Sección de divulgación científica sobre cannabis
- 🎯 Sistema de puntos de fidelidad

## 🚀 Tecnologías utilizadas

- **HTML5** - Estructura de la página
- **CSS3** - Estilos y diseño responsivo
- **JavaScript (Vanilla)** - Lógica de la aplicación
- **LocalStorage** - Persistencia de datos
